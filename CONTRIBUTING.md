# Contributing to Human Empowerment Abilities

We welcome contributions from everyone who shares our vision of cultivating human empowerment abilities in the AI era.

## How to Contribute

### 1. Add New Exercises
- Create exercises in `/exercises/` directory
- Follow the JSON format in `data/exercises.json`
- Exercises should be interactive and focused on one of the 9 abilities

### 2. Improve Documentation
- Add research references to `docs/references.md`
- Write guides for specific abilities
- Translate content into other languages

### 3. Enhance Features
- Add new interactive elements
- Improve accessibility
- Optimize performance

### 4. Report Issues
- Use GitHub Issues to report bugs
- Suggest new features
- Discuss philosophical questions

## Guidelines

### Exercise Creation Guidelines
1. Each exercise should target one specific ability
2. Include clear instructions and estimated duration
3. Add reflection prompts
4. Ensure cultural sensitivity and inclusivity

### Code Standards
1. Use semantic HTML
2. Follow existing CSS naming conventions
3. Comment complex JavaScript logic
4. Ensure mobile responsiveness

### Philosophical Alignment
All contributions should align with the project's core philosophy:
- Recognizing the non-transferable nature of human abilities
- Focusing on cultivation rather than competition
- Embracing openness and accessibility

## Getting Started
1. Fork the repository
2. Create a new branch for your contribution
3. Make your changes
4. Submit a pull request with a clear description

## Recognition
Contributors will be recognized in:
- The contributors page
- Release notes
- Community acknowledgments

Thank you for helping cultivate human empowerment!
