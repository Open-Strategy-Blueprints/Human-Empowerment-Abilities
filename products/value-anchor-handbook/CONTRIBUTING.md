# 贡献者公约

欢迎来到“价值锚点识别”知识公地！我们坚信，文明免疫系统的构建需要全球智慧的共同参与。本指南旨在确保我们的协作清晰、高效且富有建设性。

## 我们的理念
1.  **开源精神**：我们共享知识，旨在创造可复用的公共资产。
2.  **建设性协作**：我们尊重所有贡献，专注于改进提案本身。
3.  **署名荣誉**：每一位贡献者的工作都将得到恰当的记录与承认。

## 贡献方式
你可以通过以下方式参与共建：
- **修正与改进**：修复错别字、更新过时数据、优化表述。
- **案例研究**：提交使用本方法论分析的新案例（格式参考 `/case_studies/` 目录）。
- **工具开发**：开发或改进辅助识别、计算、可视化的脚本或软件。
- **翻译与本地化**：将手册翻译为其他语言版本。
- **答疑与传播**：在社区中帮助他人，分享你的应用心得。

## 工作流程
1.  **Fork 本仓库**到你自己的GitHub账户。
2.  **创建分支**：为你每项独立的改进创建一个新的特性分支。
3.  **提交更改**：在你的分支上完成修改，并编写清晰的提交说明。
4.  **发起 Pull Request (PR)**：回到本仓库，发起PR，并详细描述你的变更及其价值。
5.  **讨论与审查**：维护者和其他贡献者将对PR进行评审。请积极参与讨论。
6.  **合并**：经审核通过后，你的贡献将被合并到主分支。

## 质量要求
- **案例研究**：需遵循“四步识别法”框架，包含可验证的数据和可复现的分析过程。
- **代码工具**：需包含基本的说明文档和测试用例。
- **翻译**：需由母语者校对，确保专业术语准确。

## 成为核心维护者
持续做出高质量贡献、深刻理解项目理念的贡献者，将被邀请成为核心维护者，参与项目方向的讨论与决策。
