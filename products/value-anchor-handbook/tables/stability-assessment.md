## 📋 表格概述

### 设计目的
本表格用于系统评估候选锚点在**时间、空间、认知**三个维度上的稳定性，提供量化评分和综合稳定性等级判定。

### 适用范围
- 历史材料锚点（老照片、档案、文物）
- 文化符号锚点（汉字、图腾、仪式）
- 技艺传承锚点（手工艺、表演艺术）
- 个人/家族记忆锚点
- 城市/地标文化锚点

### 使用前准备
1. **材料收集**：收集锚点的多版本、多来源材料
2. **初步分析**：完成来源扫描，确定候选锚点
3. **证据准备**：为每个评估指标准备客观证据
4. **团队准备**：建议2-3人独立评估后取平均分

## 🔢 评估框架说明

### 评估维度与权重

#### 维度A：时间稳定性（权重40%）
评估锚点在时间维度上的持久性和一致性。

| 指标 | 评分标准 | 得分范围 | 权重 | 证据要求 |
|------|----------|----------|------|----------|
| **存在时长** | 10年以上=3分<br>50年以上=7分<br>100年以上=10分<br>按比例计算中间值 | 1-10 | 40% | 历史记录、考古证据、文献记载 |
| **形态一致性** | 变化幅度<10%=8-10分<br>10-30%=5-7分<br>>30%=1-4分<br>估算变化百分比 | 1-10 | 35% | 不同时期的形态对比 |
| **事件后存续** | 历经重大历史事件仍存续=8-10分<br>部分存续=5-7分<br>消失=1-4分<br>重建可适当加分 | 1-10 | 25% | 事件前后的对比证据 |

#### 维度B：空间稳定性（权重30%）
评估锚点在地理和文化空间中的传播和一致性。

| 指标 | 评分标准 | 得分范围 | 权重 | 证据要求 |
|------|----------|----------|------|----------|
| **地理传播范围** | 本地=1-3分<br>区域=4-6分<br>全国=7-8分<br>跨国=9-10分<br>按实际影响范围评分 | 1-10 | 60% | 分布地图、使用范围数据 |
| **跨文化理解一致性** | 理解准确率>90%=8-10分<br>70-90%=5-7分<br><70%=1-4分<br>基于测试结果 | 1-10 | 40% | 跨文化测试数据、翻译准确性 |

#### 维度C：认知稳定性（权重30%）
评估锚点在人类认知系统中的清晰度和一致性。

| 指标 | 评分标准 | 得分范围 | 权重 | 证据要求 |
|------|----------|----------|------|----------|
| **概念清晰度** | 定义明确无歧义=8-10分<br>部分明确=5-7分<br>模糊多义=1-4分<br>基于专家共识 | 1-10 | 40% | 定义文档、专家评审意见 |
| **情感唤起一致性** | 情感反应相似度>80%=8-10分<br>60-80%=5-7分<br><60%=1-4分<br>基于群体测试 | 1-10 | 35% | 情感测试数据、调查报告 |
| **记忆留存度** | 短期记忆准确率>85%=8-10分<br>70-85%=5-7分<br><70%=1-4分<br>基于记忆测试 | 1-10 | 25% | 记忆测试结果、识别率数据 |

## 🛠️ 工具使用指南

### 手动计算说明

对于不使用Python计算器的用户，可按以下步骤手动计算：

1. **计算维度得分**：
   - 时间稳定性 = (存在时长×0.40 + 形态一致性×0.35 + 事件后存续×0.25)
   - 空间稳定性 = (地理传播范围×0.60 + 跨文化理解一致性×0.40)
   - 认知稳定性 = (概念清晰度×0.40 + 情感唤起一致性×0.35 + 记忆留存度×0.25)

2. **计算综合得分**：
   - 综合稳定性 = 时间稳定性×0.40 + 空间稳定性×0.30 + 认知稳定性×0.30

3. **四舍五入规则**：所有得分保留1位小数

### 自动化计算工具

使用配套的Python计算器自动化计算：

```bash
# 进入工具目录
cd ../tools

# 交互模式（推荐初学者）
python anchor-calculator.py --interactive

# 命令行模式（快速计算）
python anchor-calculator.py \
  --duration 8.5 \
  --consistency 7.2 \
  --survival 9.0 \
  --spread 7.5 \
  --understanding 6.8 \
  --clarity 8.2 \
  --emotion 7.0 \
  --memory 7.5 \
  --name "您的锚点名称"

# 生成详细报告
python anchor-calculator.py ... --format markdown > stability_report.md
```

### 多人评估工作流程

1. **独立评估阶段**：
   - 每位评估者独立填写评估表
   - 收集客观证据支持评分
   - 记录评分理由和依据

2. **评分汇总阶段**：
   - 计算每位评估者的综合得分
   - 检查评分差异（>2分需要重新评估）
   - 计算平均分作为最终得分

3. **共识形成阶段**：
   - 讨论评分差异点
   - 重新审查争议指标
   - 形成一致的评估结论

4. **报告生成阶段**：
   - 填写综合评估部分
   - 生成稳定性证书
   - 归档所有评估材料

## ⚠️ 注意事项与常见问题

### 评估质量保证

1. **证据充分性**：每个评分都应有客观证据支持
2. **评分一致性**：相同标准应用于所有评估
3. **偏见避免**：避免时代中心主义和文化偏见
4. **记录完整性**：详细记录评分理由和依据

### 常见评估误区

| 误区 | 表现 | 纠正方法 |
|------|------|----------|
| **光环效应** | 因锚点某方面优秀而高估其他方面 | 独立评估每个指标 |
| **近期偏见** | 过分重视近期事件或变化 | 采用完整时间线视角 |
| **文化中心主义** | 用自身文化标准评判其他文化 | 进行跨文化测试 |
| **完美主义** | 对锚点要求过高，评分过低 | 采用相对标准而非绝对标准 |
| **随意评分** | 缺乏证据的主观评分 | 要求每个评分都有证据支持 |

### 特殊情况的处理

1. **缺乏历史记录**：采用间接证据，标注不确定性
2. **文化敏感性**：邀请文化背景人士参与评估
3. **新兴锚点**：降低时间权重，增加扩展性考量
4. **争议性锚点**：记录不同观点，提供多元评估

### 评估周期建议

| 锚点类型 | 初次评估 | 定期复查 | 事件触发复查 |
|----------|----------|----------|--------------|
| 高稳定性锚点（≥8.0） | 完整评估 | 每3年一次 | 重大文化事件后 |
| 中等稳定性锚点（6.0-7.9） | 完整评估 | 每2年一次 | 社会环境变化后 |
| 低稳定性锚点（4.0-5.9） | 完整评估 | 每年一次 | 使用场景变化时 |
| 实验性锚点 | 快速评估 | 每6个月 | 新技术/新需求出现 |
