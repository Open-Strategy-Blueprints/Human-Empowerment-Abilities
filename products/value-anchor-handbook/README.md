# 价值锚点识别手册：构建文明免疫的开源知识公地

> **欢迎来到一个关于“永恒价值”的开放式实验。**  
> 我们正在构建一套方法论，用以识别、评估与传承那些能跨越时间侵蚀的文化基因。这不是一个完成的产品，而是一个**正在生长的知识生态系统**的起点。我们邀请你，成为它的共同建造者。

## 🌍 项目愿景：从开放源码到开放思想

本项目的终极目标，是成为一套由全球智慧共同完善的 **“文明免疫系统”开源协议**。我们相信：
*   **真正的价值**，诞生于思想的自由流动与碰撞。
*   **知识的进化**，依赖于多元实践的验证与反馈。
*   **文明的韧性**，根植于集体对核心价值的识别与传承。

因此，我们选择将全部核心思想与方法论 **彻底开源**，并采用社区共建的“知识公地”模式运营。

## 📖 核心内容：一套可操作的系统思维工具

本手册将抽象的“文明免疫”概念，转化为可操作的 **“四步识别法”** 及配套工具，用于分析历史材料、文化符号与技艺传承中的高价值锚点。

**快速了解核心：**
- **🎯 价值锚点**：在时间、空间、认知维度上均表现稳定的高密度信息节点（如汉字、核心仪式、经典理论）。
- **🛡️ 文明免疫**：通过识别与传承锚点，文明得以在变化中保持内核稳定。
- **📚 完整手册**：阅读 [`handbook.md`](./handbook.md) 获取全部理论与方法。
- **🛠️ 工具包**：包含稳定性计算器、评估表格、练习模板等。

## 🚀 如何参与：三种角色，三种路径

无论你是谁，都能在这里找到位置。

### 👤 **路径一：作为学习者（快速体验）**
1.  **五分钟探索**：运行 `python tools/anchor-calculator.py --interactive`，体验量化分析。
2.  **完成第一个练习**：打开 [`exercises/practice-1-photos.md`](./exercises/practice-1-photos.md)，用一张老照片开始实践。
3.  **系统性学习**：按照 `handbook.md` 中的4周学习计划，逐步掌握。

### 👩💻 **路径二：作为实践者/贡献者（参与共建）**
这是本项目的心脏。我们欢迎一切能让这个知识体系变得更丰富的贡献。
- **贡献一个案例**：用本方法分析一个文化现象，提交你的分析报告。
- **改进工具或翻译**：优化计算器、制作新表格、翻译手册内容。
- **修正与深化**：修正错漏，或对某个理论点进行深入阐述。

**📘 贡献流程非常简单：**  
1.  Fork 本仓库。
2.  创建分支，进行你的改进。
3.  提交 Pull Request (PR)。
4.  参与评审讨论，等待合并。

所有详细规范，请阅读 **[CONTRIBUTING.md](./CONTRIBUTING.md)**。你的名字将永远记录在贡献者名单中。

### 🤝 **路径三：作为机构/研究者（深度应用）**
我们鼓励任何机构（博物馆、学校、企业、媒体）在遵守协议的前提下，自由地将本方法论用于：
- 内部知识管理
- 文化产品设计
- 学术研究
- **开发闭源的商业产品**

## ❓ 常见问题

**Q: 我需要付费才能学习或使用吗？**  
A: **完全不需要。** 所有知识内容永久免费开放。社区基金仅用于支付服务器等基础设施费用，并由社区透明管理。

**Q: 我没有编程或学术背景，可以贡献吗？**  
A: **当然可以。** 提交一个真实的案例分析、修正一个错别字、提出一个没讲清楚的问题，都是极其宝贵的贡献。智慧存在于多元的实践中。

**Q: 如何获得帮助或与其他贡献者交流？**  
A: 目前主要通过 **GitHub Issues 和 Discussions** 进行交流。我们期待随着社区成长，自然涌现出更多的沟通渠道。
