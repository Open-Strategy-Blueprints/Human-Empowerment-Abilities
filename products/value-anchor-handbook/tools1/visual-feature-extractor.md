# 视觉特征提取工具

## 工具简介
本工具用于对**图像类材料**（如老照片、绘画、文物图像、符号图形等）进行系统性的视觉特征识别、记录与分析，帮助从视觉信息密度高的区域发现潜在的价值锚点候选。它通过结构化表格引导观察，并辅以特征标记与比对方法，将主观视觉印象转化为可分析、可验证的客观数据。

## 使用流程
1.  **准备材料**：选定待分析的高清数字图像或实物照片。
2.  **首次浏览**：快速浏览图像，获得整体印象。
3.  **系统提取**：使用下方的**视觉特征提取记录表**，分维度扫描并记录特征。
4.  **标记与分析**：在特征清单基础上，进行特征聚类、重要性评估与锚点候选识别。
5.  **整合输出**：将提取结果整合，为后续的“稳定性评估”与“连接性映射”步骤提供输入。
